Merge Sort

What is Merge Sort? Merge sort is an algorithm that takes in an unordered array (i.e: [3,5,2,-2,16]) and puts it in order (i.e,: [-2,2,3,5,16]). It does this by recursively splitting the array in half, ordering each half, and then merging the two halves.Below is a great animation to help you out, just click MER on the top bar. Note: Click escape to get ride of the tutorial.

Part 1: argMin

What does argMin do? The argMin method will ultimately be called in the merge method. It takes in two arguments:

  • An array of arrays named a
  • An array of integers called index

Ultimately, we want to take each element from each array in a, then put them together in a single ordered array named ret. This will happen in the merge method. The argMin method performs an intermediate step. The integer at index[j] marks the place of where you are in the array a[j]; assume everything before index index[j] in a has already been added to the output array in ret. Your goal with this method is to return an index j such that the array at a[j] has the lowest integer not yet added to the merged array ret.

TLDR: In the argMin method you need to:

  1. Create a loop that checks if the index for each array in a is equal to the length of the array. If this is the true, argMin should return -1. (Think about why!) You may end up needing 2 loops to accomplish this
  2. Create a loop to find j, which is the index of a that holds the array that has the next element to be added to the merged list. This should be the j for which a[j][index[j] is the smallest. You may need to create a nested loop structure to accomplish this!
  3. Return j
NOTE: Please think carefully about indexes. Remember that all arrays start with array[0] and end with array[array.length - 1]!

Part 2: merge method

The merge method takes in one argument, which is an array of arrays named a. Your goal is to merge all of the arrays contained in a into one long array named ret. The output of merge is ret.

TLDR: In the merge method you need to:

  1. Declare a new array ret of the appropriate size
  2. Declare a new array index to keep track of your place in each array in a, and initialize it with 0s.
  3. Create a loop that:
    • Calls argMin to find which item to add to ret next
    • Adds the item to ret
    • Adds one to the appropriate integer in index
    • Stops at the appropriate time!
  4. Return ret.
