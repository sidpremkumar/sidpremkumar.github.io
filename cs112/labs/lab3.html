Lab 3

In this lab we're going to work together to solve riddles :o! We're going to get into 5 groups and each be assigned a riddle (see below) to solve. The aim is to get to be able to get the exact same outputs as seen. Before we get started lets create a class for todays lab:

  1. Open Eclipse and highlight the src folder under the Java project labs that we created in Lab 1
  2. Create a new class like we did in Lab 1
  3. Exactly like we did in Lab 1, make sure to name the file lab3, select the appropriate information and click finish.
