Modify the code so that you create a function as follows: public void performSearch(int[] a, int target). Let us understand what this all means:

  • void: void means that the output of the function is nothing. i.e., the function does not return anything.
  • performSearch: This is the name of the function.
  • int[] a: This is the first input into the function. This is an array of ints.
  • int target: This is the second input into the function. This is simply a single integer.
