public static String[] readDictionary (String dictionaryFileName) {

			fileScanner = new Scanner (new File (dictionaryFileName));
		}

			System.err.println("Currently in directory "+ System.getProperty("user.dir"));
			return null;
		}

		// read the dictionary file
		LinkedList prelimDict = new LinkedList();
		while (fileScanner.hasNext()) {
			prelimDict.add(fileScanner.next().toLowerCase());
		}
		String [] dict = prelimDict.toArray(new String[0]);
		fileScanner.close();
		if(checkAlphaOrder(dict)) {
			return dict;
		}
		else {
			System.err.println("Error: dictionary not in alphabetical order.");
			return null;
		}

The Task

Given a string s, and a dictionary of words, the purpose of FindSpacing is to return an array of all strings t that have the following properties:

  1. t is a string
  2. t is a sequence dictionary words separated by spaces
  3. t with spaces removed is the same as our input string s

In order to complete this task we will use a binary search inDictionary (which you can reuse for homework 4) method as a helper method for a recursive space method.

inDictionary Method

The purpose of the inDictionary method is to simply check if a string s is a word in a dictionary string array dict[]. Our dictionary is ordered alphabetically, and so, in order to do this we will use the binary search algorithm for sorted arrays. Here is a brief summary of inDictionary:

  • Where: inDictionary will be a private helper method, it will be used in the space method later.
  • Begin: private static boolean inDictionary(String [] dict, String s)
  • Inputs: Accepts a String s and an alphabetically ordered String[] dict
  • Algorithm: Uses the binary search algorithm to find String s in String[] dict
  • Methods: It will be important to use the compareTo method. As described in java’s API, this method compares strings alphabetically.
  • Outputs: Returns a boolean true if String s is found in String[] dict, and returns a boolean false if String s is not found in String[] dict

space Method

The purpose of the space method is to solve the task given us recursively. An outline of the method is given below:

  • Where: space will be a public method, it will be called by the driver for testing later.
  • Inputs: Accepts a String s and an alphabetically ordered String[] dict
  • Algorithm: Uses a recursive algorithm similar to what is described later in this text
  • Methods: It will be important to use the inDictionary method you wrote earlier. Also, it is helpful to be familiar with ArrayList methods as decribed in the Java API, especially the add method.
  • Outputs: Returns a String [] which includes all spaced Strings with the follwing propertieies:
    • The string is a sequence dictionary words separated by spaces
    • The string with spaces removed is the same as our input string s
